## 🛠 Development Setup

First, run the installation dependecies:

```bash
npm i
# or
yarn add package.json
```

Run the development server:

```bash
npm run dev
# or
yarn dev

```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
